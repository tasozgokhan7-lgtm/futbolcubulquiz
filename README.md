# futbolcubulquiz

**Futbolcu Bul - Quiz** mobil uygulamasının resmi tanıtım (landing) sitesi.

🔗 **Site:** https://www.futbolcubulquiz.com _(yakında yayında)_

## 📱 Uygulama
Futbol bilgi yarışması & tahmin oyunu. Sadece futbolcunun adını görürsün; kulüplerini, kupalarını, doğum yılını ve ülkesini tahmin edersin.

- **App Store:** https://apps.apple.com/tr/app/futbolcu-bul-quiz/id6767035364
- **Google Play:** https://play.google.com/store/apps/details?id=com.futbolcubul.app

## ✨ Site Özellikleri
- 🌍 Çift dilli (TR / EN) — canlı dil değiştirici
- 🎨 Koyu tema, saha yeşili tasarım, tam responsive
- 🎮 "Hemen Dene" mini oyunu (Lionel Messi) — indirmeden oyna
- 📄 Gizlilik Politikası & Destek sayfaları
- 🔍 SEO: Open Graph, Twitter Card, JSON-LD, sitemap.xml, robots.txt

## 📁 Yapı
| Dosya | Açıklama |
|-------|----------|
| `index.html` | Ana sayfa |
| `styles.css` | Tüm stiller |
| `script.js` | Dil sistemi (i18n) |
| `demo.js` | "Hemen Dene" mini oyun mantığı |
| `gizlilik.html` / `destek.html` | Yasal & destek sayfaları |
| `assets/` | İkon, ekran görüntüleri, OG görseli |

## 🚀 Yerelde Çalıştırma
Statik bir sitedir, derleme gerektirmez:

```bash
npx serve .
```
